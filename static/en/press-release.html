The Problem with AI Chat

When you tell ChatGPT to "send an email," it doesn't actually send one. When you ask it to "check my calendar," it can't. chatweb.ai changes that. 39 specialized AI agents actually send emails via Gmail, operate web browsers, execute code, publish websites, and automate workflows — all from a single natural language instruction.

What is chatweb.ai?

chatweb.ai is a multi-agent AI platform developed by Enabler, Inc. (CEO: Yuki Hamada). Users give instructions in natural language, and semantic routing automatically selects the optimal agent from 39 specialists. Multiple agents collaborate autonomously — research → analysis → report → email delivery — completed in a single prompt.

Key Features

12 AI Models

Choose from Claude Opus 4.6, Claude Sonnet, GPT-4o, Gemini 2.5 Pro, Llama 4 Scout, Kimi K2, and more. Cost per message displayed transparently. Default Llama 4 Scout costs ~$0.0002/message.

A2A Protocol v1.0

Fully compliant with Google's Agent-to-Agent protocol v1.0 (now under Linux Foundation). JSON-RPC 2.0 endpoint enables seamless interoperability with 150+ organizations in the A2A ecosystem — including ServiceNow, Salesforce, SAP, and Microsoft.

Use cases:

  • External CRM agent asks "research this lead" → chatweb.ai's Research AI executes → returns results via A2A
  • Internal chatbot asks "review this contract" → chatweb.ai's Legal AI analyzes clauses → returns risk report
  • Workflow engine asks "visualize sales data" → chatweb.ai's Analyst AI generates charts → returns report

Open Source

chatweb.ai will be open-sourced. We believe everyone should be able to build and run their own AI agent team — with full control over security, integrations, and data.

Self-Improving Feedback Loop

Every 6 hours, AI automatically analyzes quality metrics. When an agent receives 3+ negative ratings, the LLM generates and applies improved prompts — persisted across deployments. The system literally gets better while you sleep.

Quality Assurance

Plan-based quality targets (Free: 7/10, Pro: 9/10) with up to 5 improvement loops per response. Temperature, quality target, and loop limits are user-configurable.

Why We Built This

LLMs are smart enough now. MCP and API integrations have become easy. But giving a single agent every capability is genuinely hard. The most natural architecture — the one that made the most sense to me — was a team of specialists. Big tech giants are trying to do this, but nothing has shipped yet. So I built it myself. I can connect the services I actually want, verify the security with my own eyes, and iterate at my own pace. chatweb.ai will be open-sourced. I want to create a world where anyone can have their own AI agent team.

— Yuki Hamada, CEO, Enabler, Inc.
