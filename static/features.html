SITE PUBLISHING

xxx.chatweb.ai -- AIが作ってすぐ公開

「ポートフォリオサイトを作って公開して」と言うだけ。AIがHTMLを生成してCloudflare上に即時デプロイ。サブドメインは永続無料。

🌐 公開できるもの

静的HTMLからバックエンドAPI付きの動的アプリまで

  • ポートフォリオ・プロフィールサイト
  • ランディングページ・LP
  • プレゼン・スライドショー (公開リンク共有)
  • タスク管理・カレンダーアプリ
  • カウントダウンタイマー・ツール類
  • 会社紹介ページ・採用ページ
  • ミニゲーム・インタラクティブUI

⚡ システム機能 (D1 API)

各サブドメインに自動で動的APIが付与

  • GET/POST/PUT/DELETE /api/{collection} でCRUD
  • GET/POST /api/kv/:key でKey-Value保存
  • D1 SQLiteでデータ永続化 (自動テーブル作成)
  • ページビューカウンター自動記録
  • 複数アセット (CSS / JS / 画像) 対応
  • マイサイトから直接HTMLを編集・再公開
  • Cloudflare Workers + KV + D1 バックエンド
PLATFORM FEATURES

プラットフォームの主要機能

39のエージェントを支えるインフラと、あらゆるビジネスニーズに対応する機能群。

🔀

A2A マルチエージェント連携

「調べて→分析して→メールで送って」など複数ステップの依頼を自動プランニング。エージェント数に制限なく順次実行。Google A2Aプロトコル互換 (/.well-known/agent-card.json)。

🧠

長期記憶 (会話を超えて記憶)

会話内容からAIが自動で重要情報を抽出・保存。「私はエンジニアです」と伝えると次の会話でも覚えている。会話ごとの学習を蓄積。

⚙️

E2B サンドボックスコード実行

Pythonコードをセキュアなサンドボックス (E2B) で実際に実行。CSV分析、統計計算、API取得、パスワード生成などをその場で実行・検証。

🔐

暗号化シークレット管理

APIキーをFernet暗号化 (AES-128-CBC+HMAC) で安全に保存。25種以上のAPIキーに対応。ユーザー間の安全な共有機能付き。

⚡

リアルタイムストリーミング

回答が文字単位でリアルタイムに表示。処理時間とコストもリアルタイム表示。SSE (Server-Sent Events) ベースで安定接続。

🛑

生成中の停止ボタン

AIの回答生成中にいつでも停止可能。意図と違う方向に進んだ場合は即座にキャンセルしてクレジットを節約。

☰

チャット履歴 (クロスデバイス)

全会話履歴をサーバーサイドで保存。PC / スマホ / タブレットどこからでも過去の会話にアクセス可能。

💰

クレジットベース課金

月額プランでクレジットが毎月付与。追加購入も可能。フィードバック投稿で最大$1/回 (月$10上限) のリワードを獲得。

⏰

スケジュール自動化 (Cron)

「毎朝9時に天気予報をTelegramに送って」などを設定するだけで定期実行。LINE / Telegram / Webhookへの通知に対応。

⚠️

HITL (Human-in-the-Loop)

メール送信・外部API呼び出しなど実世界への影響がある操作は、AIが下書き作成後にユーザーが承認。LINE/Telegramからの承認にも対応。

📸

ブラウザ自動化 (Playwright)

WebサイトのスクリーンショットUI自動テスト・フォーム入力・スクレイピングをPlaywrightで実行。結果を画像で確認可能。

🔧

自己修復システム

フィードバックループによる自動診断。エラー発生時にAIが自律的に原因を特定し、修正を試行。継続的な品質改善。

📊

ダッシュボード・コスト分析

クレジット残高・使用状況・モデル別コスト・エージェント別利用頻度を可視化。使いすぎ防止アラート機能付き。

🌍
