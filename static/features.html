⚡ システム機能（D1 API）

各サブドメインに自動で動的APIが付与される

  • GET/POST/PUT/DELETE /api/{collection} でCRUD
  • GET/POST /api/kv/:key でKey-Value保存
  • D1 SQLiteでデータ永続化（自動テーブル作成）
  • ページビューカウンター自動記録
  • 複数アセット（CSS/JS/画像）対応
  • マイサイトから直接HTMLを編集・再公開
CORE FEATURES

プラットフォームの主要機能

エージェント群を支えるインフラとユーザー体験

🔀

A2A マルチエージェント連携

「調べてメールで送って」など複数ステップの依頼を自動的にプランニングして複数エージェントを順番に起動。最大3エージェントが協調。

🧠

長期記憶（会話を超えて記憶）

会話内容からAIが自動で重要情報を抽出・保存。「私はエンジニアです」と伝えると次の会話でも覚えている。最大50件（Free）〜無制限（Pro）。

⏰

スケジュール自動化（Cron）

「毎朝9時に天気予報をTelegramに送って」などを設定するだけで定期実行。Webhook・LINE・Telegram・Slackへの通知に対応。

⚠️

HITL（Human-in-the-Loop）

メール送信・Slack通知など実世界への影響がある操作は、AIが下書きを作成した後にユーザーが承認ボタンを押してから実行。誤送信ゼロ。

⚡

トークンストリーミング

回答が文字単位でリアルタイムに表示。待ち時間ゼロの体験。SSE（Server-Sent Events）ベースで安定した接続を維持。

📁

ファイルアップロード・解析

PDF・Excel・CSV・画像・テキストをチャット画面にドラッグ＆ドロップするだけで内容をAIが即時解析。pdfplumber + openpyxl対応。

🌍

8言語対応UI

日本語・英語・中国語・韓国語・フランス語・スペイン語・ドイツ語・ポルトガル語に完全対応。言語切り替えはワンタップ。

🔊

音声入出力（TTS/STT）

マイクボタンで音声入力、スピーカーボタンで回答を読み上げ。ハンズフリーでAIエージェントを操作できる。

📚

ナレッジベース（RAG）

PDFや文書をアップロードしてベクトル検索対応のナレッジベースを構築。社内規定・マニュアル・議事録からAIが回答。

🔧

カスタムツール作成

自社APIをchatweb.aiのツールとして登録可能。「在庫APIを呼んで」「社内DBを検索して」など独自ツールをエージェントから呼び出し。

🔗

Webhook連携

外部サービスからHTTPでトリガー可能。Slack・GitHub・Stripeなどのイベントを受け取ってAIエージェントを自動起動。

📲

LINE・Telegram連携

LINEやTelegramからchatweb.aiのエージェントに話しかけられる。スマホのメッセージアプリから全機能にアクセス。
